# Bug Report

## Title
Mixed language elements in Ukrainian interface

## Application
MeinMagenta App

## Environment
- Device: Samsung Galaxy Note 20 Ultra (SM-N986B/DS)
- OS: Android 13
- UI: One UI 5.1
- App version: 26.2519.93 (60585)
- App language: Ukrainian

## Severity
Minor

## Priority
Low

## Preconditions
App language is set to Ukrainian
User is logged into the MeinMagenta application


## Steps to Reproduce
1. Open MeinMagenta application
2. Navigate to Billing section
3. Open detailed invoice view

## Expected Result
All interface elements should be displayed in the selected language (Ukrainian). No English or German text should appearwhen Ukrainian is selected.

## Actual Result
The interface displays mixed languages simultaneously
-Ukrainian: "Інтернет", "Поповнення через 0 днів"
-English: "7,91 GB Left"
-German: "Telefonie ins EU-Ausland"

The UI does not consistenly follow the selected language
