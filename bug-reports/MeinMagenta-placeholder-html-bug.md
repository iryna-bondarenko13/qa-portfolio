# Bug Report

## Title
HTML markup displayed in billing section instead of formatted text

## Application
MeinMagenta App

## Environment
- Device: Samsung Galaxy Note 20 Ultra (SM-N986B/DS)
- OS: Android 13
- UI: One UI 5.1
- App version: 26.2519.93 (60585)
- App language: Ukrainian

## Severity
Minor

## Priority
Low

## Preconditions
User is logged into the MeinMagenta application.


## Steps to Reproduce
1. Open MeinMagenta application
2. Navigate to Billing section
3. Open detailed invoice view

## Expected Result
Billing information is displayed as properly formatted readable text.

## Actual Result
Raw HTML markup is displayed instead of formatted content.
